1、 The start and end points must be located in different zones.

2、 Priority:Fixed Price Zones > Zone to Zone > Range Prices-Downtown Area >Point to point within the zone>Range Prices-Service Area

3、 Same price for return trip or different percentage (including model increase)

4、 Program prices can be set for different car models

5、 When there is a point that is a cross zone, the price taken is determined by the setting.
